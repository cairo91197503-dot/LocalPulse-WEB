# LocalPulse Web

This is the web rewrite of the LocalPulse application, designed to run in AI Studio as a React application.
It contains the core features:
- Dashboard with Reputation Score
- AI-powered reputation diagnosis using Gemini 
- QR Code generation for Google Reviews
